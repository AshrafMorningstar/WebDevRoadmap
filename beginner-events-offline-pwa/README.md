# Local Events - Offline PWA

**Author:** Ashraf Morningstar  
**GitHub:** [https://github.com/AshrafMorningstar](https://github.com/AshrafMorningstar)

## Overview

A Progressive Web App for discovering local community events that works seamlessly offline. Features service workers, IndexedDB storage, and real-time online/offline sync.

## Key Features

- **Offline-First:** Browse events even without internet
- **Auto-Sync:** Seamlessly syncs when connection returns
- **Favorites:** Save events locally for quick access
- **Installable:** Add to home screen as native app
- **Responsive:** Perfect on mobile and desktop

## Tech Stack

- Vanilla JavaScript (ES6+)
- Service Workers for offline support
- LocalStorage for data persistence
- PWA Manifest for installability

## Setup

```bash
# Serve with any static server
npx serve .
```

## PWA Features

- Service worker caching
- Offline event browsing
- Background sync (simulated)
- Installable on mobile devices
