# Themed CSS Playground

**Author:** Ashraf Morningstar  
**GitHub:** [https://github.com/AshrafMorningstar](https://github.com/AshrafMorningstar)

## Overview

A live CSS variables playground to prototype design systems. It allows users to manipulate design tokens (colors, spacing, typography) and see real-time updates on a sample UI component. It includes responsive preview modes and token export.

## Key Features

- **Live Code Editor:** Powered by CodeMirror for syntax highlighting.
- **Instant Preview:** Isolated iframe rendering for safe style injection.
- **Theme Presets:** Quickly swap between pre-defined variable sets.
- **Responsive Testing:** Toggle between Desktop, Tablet, and Mobile views.
- **Export:** Download your finalized `theme.css` file.

## Tech Stack

- Vanilla JS
- CodeMirror (CDN)
- Iframe isolation techniques

## Usage

1. Edit the CSS variables in the left panel.
2. Observe the UI on the right change instantly.
3. Test responsiveness using the toggle buttons.
4. Click "Export Tokens" to save your work.
