# WebAssembly Image Processing Pipeline

**Author:** Ashraf Morningstar  
**GitHub:** [https://github.com/AshrafMorningstar](https://github.com/AshrafMorningstar)

## Overview

A browser-based image editor demonstrating client-side image processing. Uses Canvas API to simulate WASM-level performance for filters and transformations.

## Key Features

- **Drag & Drop:** Easy image upload
- **Real-time Filters:** Grayscale, Sepia, Blur, etc.
- **Intensity Control:** Adjustable filter strength
- **Export:** Download processed images

## Tech Stack

- Canvas API
- Vanilla JS
- CSS Filter effects
