# PWA Micro-Mentorship Platform

**Author:** Ashraf Morningstar  
**GitHub:** [https://github.com/AshrafMorningstar](https://github.com/AshrafMorningstar)

## Overview

A Progressive Web App for quick mentorship sessions. Simulates WebRTC-based video calls for 15-30 minute expert consultations.

## Key Features

- **Quick Matching:** Find available mentors instantly
- **Simulated Calls:** Mock WebRTC interface
- **PWA Ready:** Installable on mobile devices
- **Availability Status:** Real-time mentor status

## Tech Stack

- Vanilla JS
- PWA Manifest
- CSS Gradients
