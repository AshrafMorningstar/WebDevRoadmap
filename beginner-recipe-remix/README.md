# Recipe Remix - What's in My Fridge?

**Author:** Ashraf Morningstar  
**GitHub:** [https://github.com/AshrafMorningstar](https://github.com/AshrafMorningstar)

## Overview

An intelligent recipe finder that matches your available ingredients to delicious recipes. Features fuzzy matching, step-by-step cooking mode, and a beautiful culinary-focused UI.

## Key Features

- **Smart Matching:** Find recipes based on ingredients you have
- **Match Percentage:** See how well recipes match your ingredients
- **Cooking Mode:** Step-by-step guided cooking with progress tracking
- **Ingredient Chips:** Easy visual management of your ingredients
- **Responsive Design:** Perfect on mobile while cooking

## Tech Stack

- Vanilla JavaScript
- CSS with custom properties
- Local recipe database
- Fuzzy matching algorithm

## Usage

1. Add ingredients you have available
2. Click "Find Recipes"
3. Browse matching recipes sorted by match percentage
4. Click "Start Cooking" for step-by-step guidance
