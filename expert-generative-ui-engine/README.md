# Generative UI Pattern Engine

**Author:** Ashraf Morningstar  
**GitHub:** [https://github.com/AshrafMorningstar](https://github.com/AshrafMorningstar)

## Overview

A developer tool that generates production-ready UI components from natural language descriptions. This mock prototype demonstrates the "AI-to-Code" workflow.

## Key Features

- **Prompt Interface:** Natural language input
- **Mock AI Logic:** Predefined high-quality patterns
- **Live Preview:** Instant rendering with Tailwind CSS
- **Code Editor:** Manual tweaking capability
- **One-Click Copy:** Easy export

## Tech Stack

- Vanilla JS
- CodeMirror
- Tailwind CSS (CDN)
