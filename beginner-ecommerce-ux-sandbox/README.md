# Simple E-commerce UX Sandbox

**Author:** Ashraf Morningstar  
**GitHub:** [https://github.com/AshrafMorningstar](https://github.com/AshrafMorningstar)

## Overview

A functional mock-up of an e-commerce storefront focusing on User Experience (UX). It simulates a complete shopping journey: browsing products, managing a cart, and going through a secure checkout process‚Äîwithout handling real money.

## Key Features

- **Product Grid:** responsive layout with hover effects.
- **Sliding Cart:** Persistent cart state with easy addition/removal.
- **Mock Checkout:** Realistic multi-step payment form (sandbox).
- **Success State:** Rewarding confirmation screen after "purchase".

## Tech Stack

- Vanilla JS (No frameworks)
- Modern CSS (Flexbox, Grid, Backdrop Filters)

## Usage

1. Click "Add to Cart" on any product.
2. Open the cart sidebar to review items.
3. Click "Proceed to Checkout".
4. Fill in dummy data and click "Pay" to see the success animation.
