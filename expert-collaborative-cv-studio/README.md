# Collaborative Live CV Studio

**Author:** Ashraf Morningstar  
**GitHub:** [https://github.com/AshrafMorningstar](https://github.com/AshrafMorningstar)

## Overview

A real-time collaborative resume editor inspired by Google Docs. It allows multiple users (simulated) to edit a CV simultaneously. It features a rich-text editor (QuillJS) formatted effectively for A4 paper output.

## Key Features

- **Rich Text Editor:** WYSIWYG editing with headings, lists, and formatting.
- **A4 Layout:** Visual editor matches physical paper size.
- **Auto-Save:** Changes persist automatically to local storage.
- **Collaboration UI:** Shows active users and presence indicators.
- **Export Ready:** Layout designed for clean PDF generation.

## Tech Stack

- Vanilla JS
- QuillJS (Rich Text Engine)
- CSS Flexbox (Layout)

## Usage

1. Type your resume in the main editor.
2. Use the toolbar to format headers and lists.
3. Observe the "Saving..." indicator.
4. Refresh the page to see your content persisted.
