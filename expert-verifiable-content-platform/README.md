# Decentralized Content Store - Verifiable Pages

**Author:** Ashraf Morningstar  
**GitHub:** [https://github.com/AshrafMorningstar](https://github.com/AshrafMorningstar)

## Overview

A platform for publishing and verifying content authenticity using cryptographic hashing (SHA-256). Demonstrates content integrity verification without blockchain.

## Key Features

- **SHA-256 Hashing:** Generate cryptographic fingerprints
- **Content Verification:** Prove content hasn't been tampered with
- **Web Crypto API:** Browser-native cryptography
- **Trust UI:** Clear verification status

## Tech Stack

- Web Crypto API
- Vanilla JS
- SHA-256 hashing

## Usage

1. **Publish:** Enter content and generate a hash
2. **Verify:** Paste content and expected hash to verify integrity
